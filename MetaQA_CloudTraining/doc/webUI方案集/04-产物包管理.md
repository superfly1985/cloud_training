# 产物包管理模块

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **只保留必要文件** | 训练完成后保留模型文件+训练数据：best.pt、best.onnx、best_fp16.tflite、best_fp32.tflite、results.csv、dataset.yaml |
| **自动打包** | 训练+转换完成后自动打包成1个zip文件 |
| **无冗余目录** | 产物直接放在训练目录的 `package/` 子目录下，不另建models仓库 |
| **简洁命名** | zip文件名格式：`{数据集名}_v{训练序号}.zip`，训练序号从 v001 开始递增 |
| **自动清理** | 转换过程中的中间文件自动删除，只保留最终产物 |

## 2. 产物包结构

```
焊点漏包_v001.zip
├── best.pt                 # PyTorch 模型（原始训练产出）
├── best.onnx               # ONNX 格式
├── best_fp16.tflite        # TFLite FP16（移动端推荐）
├── best_fp32.tflite        # TFLite FP32（高精度）
├── results.csv             # 训练结果数据（损失曲线、指标）
├── dataset.yaml            # 原始数据集配置（类别、路径等）
└── info.json               # 产物信息
```

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|

| `training_time` | string | 训练耗时 | "1h 45m" |
| `created_at` | datetime | 创建时间 | "2026-05-11T10:30:00Z" |
| `files` | object | 包含的文件列表 |

## 7. 产物列表分页

### 7.1 分页规格

| 参数 | 值 |
|------|-----|
| 每页条数 | 12 |
| 分页方式 | 客户端分页（`filteredPackages` → `pagedPackages`） |
| 页码省略 | 超过 10 页时显示 `…`，使用全局 `calcVisiblePages()` 函数 |

### 7.2 分页条布局

```
共 N 条  [首页] [上一页] 1 2 3 ... 10 [下一页] [末页]  跳转至第 [__] 页 [跳转]
```

### 7.3 交互规则

- 搜索变更时自动重置到第 1 页
- 删除产物导致总页数减少时自动修正当前页
- 跳转输入框支持 Enter 键触发跳转
- 从训练页跳转高亮时，自动计算目标页码并跳转

## 8. 高亮跳转

### 8.1 触发方式

从训练任务页点击"查看产物"按钮跳转时，产物页接收 `highlightPackageId` 并高亮对应卡片。

### 8.2 高亮效果

- 产物卡片添加 `.highlight` 类
- CSS 动画：`@keyframes highlight-pulse`，边框颜色从 `--color-primary` 渐变到透明，持续 2 秒，循环 2 次后自动移除
- 高亮期间卡片自动滚动到可视区域（`scrollIntoView`）
- `prefers-reduced-motion` 下禁用脉冲动画，仅保留边框高亮

### 8.3 页码自动计算

高亮跳转时，根据目标产物在 `filteredPackages` 中的索引自动计算所在页码：

```javascript
var idx = this.filteredPackages.findIndex(function(p) { return p.id === id; });
if (idx >= 0) {
  this.pkgPage = Math.floor(idx / this.pkgPageSize) + 1;
}
```

## 5. 产物包命名规则

**格式**：`{数据集名}_v{训练序号}.zip`

| 训练次数 | 产物包名称 |
|---------|-----------|
| 第1次 | `焊点漏包_v001.zip` |
| 第2次 | `焊点漏包_v002.zip` |
| 第3次 | `焊点漏包_v003.zip` |

> **注意**：训练序号自动递增，不依赖数据集变化。每次训练都会生成新的v号。

## 6. 核心 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/packages` | GET | 产物包列表 |
| `/api/v1/packages/{package_id}` | GET | 产物包详情 |
| `/api/v1/packages/{package_id}/download` | GET | 下载产物包 |
| `/api/v1/packages/{package_id}` | DELETE | 删除产物包 |
| `/api/v1/models/convert` | POST | 手动触发转换 |
