# 前端界面优化方案

本文档汇总前端界面的 5 项优化需求，明确修改范围与实现细节，作为后续编码实施的依据。

## 背景

当前前端已实现 Grafana 风格 Dashboard 布局，包含概览、数据集、训练、产物、系统 5 个功能页。经过初步使用，发现数据集浏览体验、全局搜索能力、训练到产物的跳转链路、品牌标识及版本展示等方面存在改进空间。

---

## 需求一：数据集页面 — 图片列表与预览区重构

### 目标

解决图片列表过长导致预览区被撑出屏幕、列表宽度浪费、缺少多选删除和页码跳转等问题，提升数据集浏览效率。

### 具体修改项

#### 1.1 布局重构

- `.dataset-layout` 改为固定高度 `height: 500px`，禁止整体被撑开。
- 左侧图片列表面板独立滚动：`.dataset-list-body` 设置 `overflow-y: auto`，高度占满剩余空间。
- 右侧预览区始终可见：不随左侧列表滚动，保持固定展示区域。
- 宽度比例调整为 `grid-template-columns: 2fr 3fr`（列表 40%，预览 60%）。

#### 1.2 文件名截断

- 新增 `truncateFilename` 方法。
- 规则：保留前 6 个字符 + `***` + 后 3 个字符（不含扩展名）。
- 示例：`image_000001.jpg` 显示为 `image_***001`。
- 完整文件名在鼠标悬停时通过 `title` 属性展示。

#### 1.3 多选删除

- 新增 `selectedImageIds: []` 数组维护选中状态。
- 交互方式：
  - 每张图片项左侧增加复选框（checkbox）。
  - 支持 `Ctrl + 单击` 多选，`Shift + 单击` 连续多选。
  - 顶部增加"全选/取消全选"复选框。
- 删除按钮位置：与分页栏同排右侧，显示"删除选中 (N)"，仅当 `selectedImageIds.length > 0` 时可用。
- 删除前调用根实例的 `confirm` 弹窗确认。

#### 1.4 分页增强

- 分页栏布局：上一页 | 页码按钮 1 2 3 ... 10 | 下一页 | 跳转至第 [__] 页 [跳转]。
- 页码按钮最多显示 10 个连续页码，超出用 `...` 省略。
- 当前页码高亮显示。
- 删除按钮放在分页栏右侧同排。

#### 1.5 预览区优化

- 预览区显示固定比例的图片占位（4:3 或 16:9 自适应）。
- 当前选中图片信息：文件名、尺寸、大小、标注状态。
- 未选中时显示占位提示："选择图片预览"。

### 涉及文件

- `js/dataset-tab.js`：模板重构、多选逻辑、分页逻辑、文件名截断方法。
- `css/style.css`：`.dataset-layout`、`.dataset-list-panel`、`.preview-panel`、分页栏、复选框样式。

## 需求二：全局快速搜索

### 目标

在数据集、训练任务、产物包三个页面增加动态模糊搜索，方便在数据量大时快速定位目标项。

### 具体修改项

#### 2.1 搜索框设计

- 位置：每个页面的 `toolbar` 区域左侧（按钮之前）。
- 样式：带 `bi-search` 图标的输入框，宽度约 240px，`border-radius: var(--radius-md)`。
- Placeholder：
  - 数据集页："搜索数据集名称…"
  - 训练页："搜索数据集或版本…"
  - 产物页："搜索产物名称…"

#### 2.2 搜索逻辑

- 采用动态筛选：输入时实时过滤，不调用后端接口。
- 搜索字段：
  - 数据集页：匹配 `dataset.name`。
  - 训练页：匹配 `task.dataset_name` 或 `task.version`。
  - 产物页：匹配 `package.name` 或 `package.dataset_name`。
- 实现方式：使用 `computed` 属性 `filteredItems`，基于 `searchQuery` 做 `String.includes()` 模糊匹配。
- 搜索无结果时显示空状态提示："未找到匹配项"。

#### 2.3 训练页额外筛选

- 搜索框旁增加状态下拉筛选器：全部 / 运行中 / 已完成 / 已停止 / 失败。
- 搜索与状态筛选叠加生效。

### 涉及文件

- `js/dataset-tab.js`：搜索框 + `filteredDatasets` computed。
- `js/training-tab.js`：搜索框 + 状态筛选 + `filteredTasks` computed。
- `js/package-tab.js`：搜索框 + `filteredPackages` computed。
- `css/style.css`：搜索框样式、筛选器布局。

## 需求三：训练页面 — 跳转到产物卡片

### 目标

在训练任务的操作栏增加"查看产物"入口，支持从已完成的训练任务一键跳转到产物包页面并高亮对应卡片。

### 具体修改项

#### 3.1 操作栏新增按钮

- 图标：`bi-box-seam`（与产物页图标一致）。
- 位置：查看日志、损失曲线之后，停止训练之前。
- 显示条件：仅当 `task.status === 'completed'` 时显示。
- `aria-label` 和 `title` 均为"查看产物"。

#### 3.2 跳转逻辑

- 点击后通过 `task.dataset_name` + `task.version` 在产物列表中查找对应产物包。
- 调用 `this.$root.navigateTo('package')` 切换标签页。
- 通过根实例传递 `highlightPackageId`，产物页接收后给对应卡片添加高亮状态。
- 若未找到对应产物包，弹出提示"未找到对应产物包"。

#### 3.3 高亮效果

- 产物卡片添加 `.highlight` 类。
- CSS 动画：`@keyframes highlight-pulse`，边框颜色从 `--color-primary` 渐变到透明，持续 2 秒，循环 2 次后自动移除。
- 高亮期间卡片自动滚动到可视区域（`scrollIntoView`）。

### 涉及文件

- `js/training-tab.js`：新增按钮 + 跳转逻辑。
- `js/package-tab.js`：接收 `highlightPackageId` + 高亮动画触发。
- `js/app.js`：增加 `highlightPackageId` 状态传递。
- `css/style.css`：`.highlight` 动画样式。

## 需求五：侧边栏左下角显示版本号

### 目标

### 具体修改项

  - `color: var(--color-text-muted)`。

### 涉及文件

- `index.html`：侧边栏底部 HTML。
- `css/style.css`：`.sidebar-footer` 样式。

## 需求六：三个列表页客户端分页

### 目标

数据集表格、训练任务列表、产物包列表在数据量大时一次加载所有条目，影响性能和浏览体验。增加客户端分页，每页显示固定条数。

### 具体修改项

#### 6.1 数据集表格分页

- 每页 **8 条**（因上下分页布局，8 条视觉更合理）。
- 新增 `dsPage`、`dsPageSize`、`dsJumpPage` 状态变量。
- 新增 `filteredDatasets`（搜索过滤）、`pagedDatasets`（分页切片） computed 属性。
- 模板中 `v-for` 从 `datasets` 改为 `pagedDatasets`。
- 分页条：共 N 条 + 首页/上一页/页码/下一页/末页 + 跳转输入框。
- 搜索变更时重置 `dsPage = 1`。

#### 6.2 训练任务列表分页

- 每页 **15 条**。
- 新增 `taskPage`、`taskPageSize`、`taskJumpPage` 状态变量。
- 新增 `filteredTasks`（搜索 + 状态筛选）、`pagedTasks`（分页切片） computed 属性。
- 模板中 `v-for` 从 `tasks` 改为 `pagedTasks`。
- 分页条同上布局。
- 搜索或状态筛选变更时重置 `taskPage = 1`。

#### 6.3 产物包列表分页

- 每页 **12 条**。
- 新增 `pkgPage`、`pkgPageSize`、`pkgJumpPage` 状态变量。
- 新增 `filteredPackages`（搜索过滤）、`pagedPackages`（分页切片） computed 属性。
- 模板中 `v-for` 从 `packages` 改为 `pagedPackages`。
- 分页条同上布局。
- 高亮跳转时自动计算目标页码：`Math.floor(idx / pkgPageSize) + 1`。

#### 6.4 全局页码计算函数

- 在 `app.js` 中新增 `calcVisiblePages(current, total)` 全局函数。
- 总页数 ≤ 10 时显示全部页码。
- 超过 10 页时显示当前页附近 ±4/±5 范围，首尾页始终显示，中间用 `…` 省略。
- 三个页面共用此函数，通过 `dsVisiblePages`、`taskVisiblePages`、`pkgVisiblePages` computed 调用。

#### 6.5 运行中状态 Spinner 修复

- CSS 选择器从 `.btn .spinner` 改为通用 `.spinner`，确保状态徽章内的 spinner 也能显示。
- 新增 `.badge .spinner` 子选择器设置小尺寸（10px）和右边距（4px）。
- 模板中 spinner 的内联样式移除，统一由 CSS 控制。

### 涉及文件

- `js/dataset-tab.js`：分页状态、computed、分页条模板。
- `js/training-tab.js`：分页状态、computed、分页条模板、spinner 内联样式移除。
- `js/package-tab.js`：分页状态、computed、分页条模板、高亮跳转页码计算。
- `js/app.js`：`calcVisiblePages()` 全局函数。
- `css/style.css`：`.spinner` 通用化、`.badge .spinner` 子选择器。

## 验收标准

- 数据集图片列表固定高度 500px，左侧独立滚动，右侧预览区始终可见。
- 文件名截断格式统一为 `前6***后3`。
- 多选删除支持 checkbox 和快捷键，删除前需确认。
- 分页栏包含页码按钮和跳转输入框。
- 三个页面搜索框实时模糊过滤，训练页支持状态叠加筛选。
- 已完成训练任务的操作栏出现"查看产物"按钮，点击后跳转并高亮对应产物卡片。
- 所有"YOLO"品牌标识替换为"YOLO云算"。
- 侧边栏左下角显示版本号，展开/折叠状态自适应。
- 数据集表格每页 8 条，训练任务列表每页 15 条，产物包列表每页 12 条。
- 三个列表页分页条布局一致：共 N 条 + 首页/上一页/页码/下一页/末页 + 跳转。
- 搜索/筛选变更时自动重置到第 1 页。
- 运行中状态显示旋转 Spinner 图标，CSS 选择器通用化。
- 高亮跳转时自动计算目标页码并跳转。
- 所有新增交互保持 `aria-*` 无障碍属性完整。
- `prefers-reduced-motion` 下禁用高亮脉冲动画。
