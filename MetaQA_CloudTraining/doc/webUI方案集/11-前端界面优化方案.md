# 前端界面优化方案

本文档记录当前版本已经落地的前端界面优化结果，作为 `static/` 目录下
实现现状的说明。以下内容已不是待开发需求，而是当前版本的实际行为。

## 1. 当前优化范围

当前 Web 前端已经完成以下几类界面优化：

- 数据集页搜索、分页、多选删除和预览区重构
- 训练页搜索、状态筛选和“查看产物”跳转
- 产物页搜索、分页和高亮定位
- 品牌名统一为 `YOLO云算`
- 侧边栏底部显示版本号
- 系统页补充环境修复任务进度和日志下载

## 2. 数据集页

当前数据集页主要由 `static/js/dataset-tab.js` 和
`static/css/style.css` 驱动，已经实现以下能力。

### 2.1 搜索与分页

- 顶部提供搜索框，使用 `searchQuery` 实时过滤数据集名称
- 搜索无结果时显示“未找到匹配项”
- 列表分页由客户端完成
- 页码显示逻辑复用全局 `calcVisiblePages()`

### 2.2 图片列表与预览区

- 左侧图片列表支持独立滚动
- 右侧预览区固定显示，不随列表滚动丢失
- 未选择图片时显示占位提示
- 选中图片后显示预览与标注信息

### 2.3 多选删除

当前版本已经支持以下多选操作：

- 单个复选框选中
- 顶部全选/取消全选
- `Ctrl + 单击` 多选
- `Shift + 单击` 区间多选
- “删除选中 (N)” 按钮联动选中数量
- 删除前弹出确认对话框

### 2.4 相关状态

当前实现中可见的关键状态包括：

- `searchQuery`
- `selectedImageIds`
- `dsPage`
- `imgPage`

## 3. 训练页

当前训练页主要由 `static/js/training-tab.js` 实现。

### 3.1 搜索与筛选

- 搜索框支持按数据集名称和版本号过滤
- 状态下拉框支持叠加筛选
- 搜索和筛选变化时自动回到第一页

### 3.2 分页

- 训练任务列表采用客户端分页
- 页码显示与其他列表页保持统一

### 3.3 跳转到产物

当前版本已实现“查看产物”跳转链路：

1. 训练任务点击“查看产物”
2. 根实例写入 `highlightPackageId`
3. 调用 `navigateTo("package")`
4. 产物页定位并高亮对应卡片

## 4. 产物页

当前产物页主要由 `static/js/package-tab.js` 实现。

### 4.1 搜索与分页

- 提供产物名称搜索框
- 搜索结果为空时显示空状态提示
- 卡片列表使用客户端分页

### 4.2 高亮定位

产物页会读取根实例上的 `highlightPackageId`：

- 自动切换到目标页
- 滚动到对应卡片
- 应用高亮样式
- 使用后清空高亮状态，避免重复触发

## 5. 全局交互与基础设施

当前全局前端能力主要由 `static/js/app.js` 提供。

### 5.1 全局页码计算

当前版本已经提供 `calcVisiblePages(current, total)`，
供数据集页、训练页、产物页共用页码显示逻辑。

### 5.2 全局状态

当前根实例里已经存在以下关键状态：

| 字段 | 用途 |
|------|------|
| `highlightPackageId` | 跨页面高亮产物卡片 |
| `appVersion` | 侧边栏底部展示版本号 |
| `envFixing` | 顶部环境状态显示“修复中” |
| `envHasFailures` | 顶部环境状态显示异常 |

## 6. 品牌与版本展示

当前品牌名已经统一改为 `YOLO云算`。

### 6.1 已落地位置

| 位置 | 当前实现 |
|------|----------|
| 页面标题 | `YOLO云算 - 云端训练平台` |
| 页面描述 | `YOLO云算 - 云端训练平台 - 数据集管理、模型训练、产物包管理` |
| 侧边栏 Logo | 展开时显示 `YOLO云算` |
| 概览页文案 | 显示 `YOLO云算运行状态总览` |

### 6.2 版本号展示

当前版本号展示规则如下：

- 根实例 `appVersion` 当前值为 `0.1.0`
- 展开侧边栏时显示 `v0.1.0`
- 折叠侧边栏时显示 `0.1`
- 样式由 `.sidebar-footer` 控制

注意，仓库配置文件中的应用版本号当前为 `v0.1.0`，前端显示值仍然来自
`app.js` 中的 `0.1.0`。

## 7. 系统页补充优化

系统页已不只是展示环境检查结果，还集成了修复任务视图。

当前已落地内容包括：

- 修复任务状态徽章
- 当前步骤名称
- 修复进度条
- 两列步骤列表
- 最近日志滚动区域
- 日志下载按钮
- 修复中按钮禁用与轮询恢复

## 8. 相关文件

当前前端优化主要落在以下文件中。

| 文件 | 当前职责 |
|------|----------|
| `static/js/dataset-tab.js` | 数据集搜索、分页、多选删除、预览区交互 |
| `static/js/training-tab.js` | 训练列表搜索、筛选、产物跳转 |
| `static/js/package-tab.js` | 产物搜索、分页、高亮定位 |
| `static/js/app.js` | 根实例状态、全局页码计算、导航 |
| `static/js/system-tab.js` | 环境检查、自动修复任务视图 |
| `static/index.html` | 品牌名、侧边栏版本号、主布局 |
| `static/css/style.css` | 侧边栏底部、卡片和列表样式 |

## 9. 当前验收结果

- 数据集页支持搜索、分页和多选删除
- 训练页支持搜索、状态筛选和产物跳转
- 产物页支持搜索、分页和高亮定位
- 品牌名已统一为 `YOLO云算`
- 侧边栏底部已显示版本号
- 系统页已支持修复任务进度和日志下载
