# 固定域名访问方案

> 针对训练服务器IP变化，提供固定域名访问的解决方案

---

## 问题背景

- 训练服务器IP经常变化（动态IP）
- 需要固定网址访问WebUI（如 `http://train.meta-qa.cloud:8090`）
- 已有域名：`meta-qa.cloud`
- DNS服务商：DNSPod（腾讯云）

---

## 解决方案

### 方案一：DDNS动态域名解析（推荐）

在训练服务器上安装DDNS客户端，IP变化时自动更新DNS解析。

#### 1. 获取DNSPod API密钥

1. 登录 https://www.dnspod.cn/
2. 账号中心 → API密钥管理
3. 创建密钥（DNSPod Token）
4. 保存 **ID** 和 **Token**

#### 2. 安装DDNS客户端（ddns-go）

```bash
# 下载安装
wget https://github.com/jeessy2/ddns-go/releases/download/v6.0.0/ddns-go_6.0.0_linux_x86_64.tar.gz
tar -xzf ddns-go_6.0.0_linux_x86_64.tar.gz
sudo mv ddns-go /usr/local/bin/

# 启动服务
sudo ddns-go -s install
sudo ddns-go -s start
```

#### 3. 配置DDNS

访问 `http://训练服务器IP:9876`

| 配置项 | 值 |
|--------|-----|
| DNS服务商 | DNSPod |
| ID | 你的Token ID |
| Token | 你的Token密钥 |
| 域名 | train.meta-qa.cloud |

#### 4. 访问方式

```
http://train.meta-qa.cloud:8090
```

---

### 方案二：内网穿透（备用）

无需公网IP，使用第三方服务提供固定域名。

| 工具 | 免费域名 | 说明 |
|------|---------|------|
| **花生壳** | 有 | 国内老牌，稳定 |
| **Ngrok** | 有 | 国外服务，简单 |
| **Frp** | 需自建 | 需要自己部署服务端 |

#### 花生壳配置示例

1. 注册花生壳账号
2. 购买/领取免费域名
3. 安装花生壳客户端
4. 配置映射：外网域名 → 内网 `localhost:8090`

---

### 方案三：VPN组网（内网访问）

将训练服务器加入虚拟局域网，分配固定虚拟IP。

| 工具 | 特点 |
|------|------|
| **ZeroTier** | 免费，支持100台设备 |
| **Tailscale** | 基于WireGuard，简单易用 |
| **WireGuard** | 自建，完全可控 |

#### ZeroTier配置示例

```bash
# 安装
curl -s https://install.zerotier.com | sudo bash

# 加入网络（需先在ZeroTier控制台创建网络）
sudo zerotier-cli join <Network ID>

# 查看虚拟IP
sudo zerotier-cli listnetworks
```

访问方式：
```
http://虚拟IP:8090
```

## 方案对比

| 方案 | 成本 | 复杂度 | 稳定性 | 适用场景 |
|------|------|--------|--------|---------|
| **DDNS** | 免费 | 低 | 高 | 有域名，会配置DNS |
| **内网穿透** | 免费/付费 | 低 | 中 | 无公网IP，快速部署 |
| **VPN组网** | 免费 | 中 | 高 | 多台服务器组网访问 |

---

## 推荐配置

**首选方案一（DDNS）**，原因：
1. 已有域名 `meta-qa.cloud`
2. DNSPod是国内服务商，速度快
3. 配置简单，一次设置长期有效
4. 访问速度最快（直接连接公网IP）
