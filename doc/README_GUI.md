# 云端训练脚本优化可视化GUI界面

## 功能概述

这是一个用于云端训练脚本优化的可视化GUI界面，提供了完整的云端YOLO模型训练管理功能。

## 核心功能模块

### 1. 服务器配置模块
- **服务器连接设置**: 支持IP地址、端口、用户名、密码配置
- **SSH密钥支持**: 可选择使用SSH密钥文件进行安全连接
- **连接测试**: 实时测试服务器连接状态
- **配置保存**: 自动保存服务器配置信息

### 2. 训练数据集配置模块
- **本地数据集选择**: 支持选择本地YOLO数据集目录
- **数据集分析**: 自动解析dataset.yaml文件，获取类别信息
- **训练脚本生成**: 根据数据集和参数自动生成训练脚本
- **多线程上传**: 支持断点续传功能，高效上传数据到云端
- **云端数据管理**: 提供清理云端数据集和脚本的功能

### 3. 训练监控模块
- **训练参数配置**: 支持设置训练轮数、批次大小、学习率等参数
- **模型选择**: 支持多种YOLO模型（yolov8n/s/m/l/x）
- **训练控制**: 提供开始训练、停止训练、监控状态等功能
- **实时日志**: 显示训练过程中的实时日志信息
- **模型下载**: 自动下载训练完成的最佳模型

## 安装和使用

### 环境要求
- Python 3.8+
- Windows操作系统
- 网络连接（用于连接云端服务器）

### 安装步骤

1. **安装依赖包**:
   ```bash
   pip install -r requirements_gui.txt
   ```

2. **启动GUI应用**:
   - 方式1: 双击运行 `start_gui.bat`
   - 方式2: 命令行运行 `python cloud_training_gui.py`

### 使用流程

#### 第一步：配置服务器连接
1. 在"服务器配置"选项卡中填写服务器信息
2. 点击"测试连接"验证连接状态
3. 点击"保存配置"保存服务器信息

#### 第二步：配置数据集
1. 在"数据集配置"选项卡中选择本地数据集路径
2. 系统会自动分析数据集结构和类别信息
3. 点击"生成训练脚本"创建训练脚本
4. 点击"上传数据集"将数据上传到云端

#### 第三步：开始训练和监控
1. 在"训练监控"选项卡中设置训练参数
2. 点击"开始训练"启动云端训练
3. 点击"开始监控"实时监控训练状态
4. 训练完成后点击"下载模型"获取训练结果

## 界面特性

### 现代化设计
- 采用ttk组件，提供现代化的界面风格
- 清晰的选项卡布局，功能模块分离明确
- 实时状态显示和进度条反馈

### 智能化功能
- 自动保存所有配置信息
- 智能数据集分析和脚本生成
- 多线程文件上传，支持断点续传
- 实时训练状态监控

### 安全性保障
- 支持SSH密钥认证
- 密码输入隐藏显示
- 连接状态实时验证

## 配置文件

应用会自动创建 `cloud_training_config.json` 配置文件，包含：
- 服务器连接信息
- 数据集配置参数
- 训练参数设置

所有配置信息会自动保存，下次启动时自动加载。

## 日志系统

应用提供完整的日志记录功能：
- 界面内实时日志显示
- 自动保存到 `cloud_training_gui.log` 文件
- 包含时间戳和详细操作信息

## 故障排除

### 常见问题

1. **连接失败**
   - 检查服务器IP地址和端口是否正确
   - 确认用户名和密码是否正确
   - 检查网络连接是否正常

2. **上传失败**
   - 确认服务器连接正常
   - 检查远程路径权限
   - 确认本地数据集路径有效

3. **训练启动失败**
   - 确认数据集已成功上传
   - 检查训练参数设置
   - 查看服务器端Python环境

### 技术支持

如遇到问题，请查看：
1. 界面内的实时日志信息
2. `cloud_training_gui.log` 日志文件
3. 服务器端的训练日志
