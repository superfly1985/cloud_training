## 下载说明

### 方法1: 直接下载
1. 复制上述URL到浏览器下载
2. 将下载的.whl文件放入 `D:\OneDrive\24.Visual AI\training_scripts\Environment_package\` 目录

### 方法2: 使用pip download命令
```bash
# 创建下载目录
mkdir -p "D:\OneDrive\24.Visual AI\training_scripts\Environment_package"
cd "D:\OneDrive\24.Visual AI\training_scripts\Environment_package"

# 下载PyTorch包
pip download torch==2.0.1 torchvision==0.15.2 torchaudio==2.0.2 --index-url https://download.pytorch.org/whl/cpu

# 下载其他包
pip download ultralytics==8.0.196 networkx==2.8.8 numpy opencv-python pillow matplotlib scipy pandas requests tqdm pyyaml seaborn psutil thop filelock jinja2 sympy typing-extensions setuptools wheel
```

### 方法3: 使用requirements文件
创建requirements.txt文件:
```
torch==2.0.1 --index-url https://download.pytorch.org/whl/cpu
torchvision==0.15.2 --index-url https://download.pytorch.org/whl/cpu
torchaudio==2.0.2 --index-url https://download.pytorch.org/whl/cpu
ultralytics==8.0.196
networkx==2.8.8
numpy
opencv-python
pillow
matplotlib
scipy
pandas
requests
tqdm
pyyaml
seaborn
psutil
thop
filelock
jinja2
sympy
typing-extensions
setuptools
wheel
```

然后运行:
```bash
pip download -r requirements.txt -d "D:\OneDrive\24.Visual AI\training_scripts\Environment_package"
```

## 注意事项

1. **平台兼容性**: 上述URL包含Windows和Linux版本，请根据目标平台选择
2. **Python版本**: 主要针对Python 3.8，如果使用其他版本请修改cp38为对应版本
3. **架构**: 包含x86_64和win_amd64架构，请根据系统选择
4. **版本锁定**: 为了兼容性，建议使用指定版本而非最新版本
5. **依赖关系**: 某些包可能有额外依赖，下载时会自动包含
