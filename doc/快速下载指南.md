# 快速下载指南

## 方法一：使用批处理脚本 (推荐)

### Windows用户
双击运行 `manual_download_commands.bat` 文件，或在命令行中执行：
```cmd
manual_download_commands.bat
```

### Linux/Mac用户
在终端中执行：
```bash
chmod +x manual_download_commands.sh
./manual_download_commands.sh
```

## 方法二：使用requirements文件

### 下载核心包 (最小化)
```bash
pip download -r requirements_core.txt -d Environment_package/
```

### 下载完整包 (包含所有可能用到的包)
```bash
pip download -r requirements_offline.txt -d Environment_package/
```

## 方法三：手动逐个下载

### 1. 创建下载目录
```bash
mkdir -p "Environment_package"
cd "Environment_package"
```

### 2. 下载核心包
```bash
# PyTorch
pip download torch==2.0.1 torchvision==0.15.2 torchaudio==2.0.2 --index-url https://download.pytorch.org/whl/cpu

# Ultralytics
pip download ultralytics==8.0.196

# 其他核心包
pip download networkx==2.8.8 numpy opencv-python pillow matplotlib requests tqdm pyyaml filelock jinja2 sympy typing-extensions setuptools wheel
```

## 方法四：直接下载链接

参考 `package_download_urls.md` 文件中的直接下载链接。

## 下载后的使用

### 1. 验证下载
```bash
ls -la Environment_package/
```

### 2. 离线安装
```bash
# 安装单个包
pip install Environment_package/torch-2.0.1+cpu-cp38-cp38-win_amd64.whl

# 安装目录中的所有包
pip install Environment_package/*.whl

# 从本地目录安装
pip install --find-links Environment_package/ torch torchvision torchaudio ultralytics
```

## 注意事项

1. **网络环境**: 如果网络较慢，建议分批下载
2. **存储空间**: 完整下载大约需要2-3GB空间
3. **Python版本**: 确保下载的包与目标Python版本兼容
4. **平台兼容**: Windows和Linux包不通用，注意选择正确平台

## 推荐下载顺序

1. **第一优先级** (必需):
   - PyTorch (torch, torchvision, torchaudio)
   - Ultralytics
   - NetworkX
   - NumPy, OpenCV, Pillow

2. **第二优先级** (重要):
   - Matplotlib, Requests, TQDM, PyYAML
   - PyTorch依赖包 (filelock, jinja2, sympy, typing-extensions)

3. **第三优先级** (可选):
   - 机器学习包 (scikit-learn, scipy, pandas)
   - 实验跟踪包 (tensorboard, wandb)

## 故障排除

### 下载失败
- 检查网络连接
- 尝试更换pip镜像源
- 分批下载，避免一次性下载过多

### 包冲突
- 使用虚拟环境
- 检查Python版本兼容性
- 使用指定版本而非最新版本

### 空间不足
- 清理临时文件
- 只下载核心包
- 使用外部存储设备
