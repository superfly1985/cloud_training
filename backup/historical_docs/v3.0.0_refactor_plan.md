# v3.0.0 重构计划

## 1. 重构目标

- **版本升级**: v2.2.10 → v3.0.0 (重大版本)
- **核心变更**: UI布局重新设计为3行3列网格
- **迁移策略**: 逐步迁移，原程序保持不动

## 2. 迁移策略

### 2.1 双版本并行

```
云端训练/
├── cloud_training_gui.py    # 原版本 (v2.2.10，保持完整)
├── main_v3.py               # 新版本入口 (v3.0.0，逐步完善)
├── src/
│   ├── core/                # 业务逻辑模块
│   ├── ui/                  # UI组件模块
│   ├── utils/               # 工具函数
│   └── models/              # 数据模型
└── docs/
    └── v3.0.0_refactor_plan.md  # 本计划文档
```

### 2.2 实施步骤

| 阶段 | 任务 | 状态 | 说明 |
|------|------|------|------|
| 1 | 创建新入口 main_v3.py | 待实施 | 新版本主程序 |
| 2 | 实现3行3列UI布局 | 待实施 | 按设计稿布局 |
| 3 | 复用原业务逻辑 | 待实施 | 从原文件复制/导入 |
| 4 | 功能测试验证 | 待实施 | 对比原版本功能 |
| 5 | 文档更新 | 待实施 | README、changelog |

## 3. UI布局设计

### 3.1 新布局 (3行3列)

```
┌─────────────────────────────────────────────────────────────────┐
│ 云端训练管理平台 v3.0.0 (标题栏)                                    │
├───────────────┬──────────────────────────────┬──────────────────┤
│               │                              │                  │
│  服务器设置    │         系统监控              │      训练日志     │
│  (左上)       │         (中上)                │      (右上)      │
│  280px        │         flex                  │      380px       │
│               │                              │                  │
├───────────────┴──────────────────────────────┴──────────────────┤
│                                                                  │
│   训练参数              │  操作进度                               │
│   (左中)               │  (右中)                                │
│   flex                 │  flex                                  │
│                                                                  │
├────────────────────────┬─────────────────────┬──────────────────┤
│                        │                     │                  │
│   图像增强              │   数据集信息          │   快捷功能卡片    │
│   (左下)               │   (中下)             │   (右下)         │
│   300px                │   flex               │   280px          │
│                        │                     │                  │
└────────────────────────┴─────────────────────┴──────────────────┘
```

### 3.2 各区域内容

| 区域 | 位置 | 宽度 | 内容 |
|------|------|------|------|
| 服务器设置 | 左上 | 280px | IP、端口、用户名、密码、密钥、按钮 |
| 系统监控 | 中上 | flex | CPU、内存、GPU、磁盘图表 |
| 训练日志 | 右上 | 380px | 实时日志输出 |
| 训练参数 | 左中 | flex | epochs、batch、lr、imgsz、模型选择 |
| 操作进度 | 右中 | flex | 进度条、状态显示 |
| 图像增强 | 左下 | 300px | 5个滑块(scale/fliplr/hsv_h/hsv_s/hsv_v) |
| 数据集信息 | 中下 | flex | 本地/云端数据集状态 |
| 快捷功能 | 右下 | 280px | 3个竖向卡片 |

## 6. 回滚方案

如果新版本有问题，随时可回滚：

```bash

python cloud_training_gui.py

# 运行新版本
python main_v3.py
```

两个版本完全独立，互不影响。
